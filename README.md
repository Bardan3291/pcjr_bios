## 🛠️ Using the Software

The files you downloaded contain the heartbeat of the IBM PCjr. These are called ROM files. An emulator uses these files to start a virtual version of the original computer.

1. Open your emulator program.
2. Locate the settings menu or the configuration file.
3. Find the option labeled BIOS Path or ROM Path.
4. Point this path to the folder where you kept the extracted files from this project.
5. Restart your emulator.
6. The software will load the identical code that IBM used in their original hardware.

## 📂 Understanding the Files

This project organizes data into specific file types. You will see several files in your folder.

- .BIN files: These contain the actual code the computer reads during startup.
- .INC files: These act as supporting notes that help the build process match the original IBM specifications.
- .ASM files: These represent the text version of the code. If you know how to read computer language, you can open these in a simple text editor like Notepad.

## 💡 Troubleshooting

- Version mismatch: Use the files provided in the latest release link. Older versions might lack the required parts for your emulator.

## 📝 Frequently Asked Questions

What if I want to change the code?
You are free to open the text files and make adjustments. The project provides the foundation, but you own the copy on your machine. Be aware that changing these files might prevent the software from starting in an emulator.

Can this run on actual vintage hardware?
Yes. If you have a device that can burn data onto physical ROM chips, you can use these files to replace a faulty chip in an original IBM PCjr.

Is this software safe?
The files are plain text and data code. They contain no viruses or harmful programs. They simple provide the instructions that a vintage computer needs to turn on. 

Where can I find more help?
Your emulator software usually has a documentation file. If you cannot find the BIOS setting, search the help menu of your emulator for the word BIOS. Most modern emulators have simple menus designed for this purpose.

Does this include games?
This project only includes the basic system software. It does not include games or extra programs. It creates the environment needed to run those items on your computer. You must obtain game files separately to play them through the system.
